ЛАБОРАТОРНАЯ РАБОТА №4а (дополнительная)   
Задание  
Создайте класс Text, который должен обрабатывать входной текст.
В классе обязательно должны быть два поля: указатель на динамический массив строк и размер массива. 
Реализуйте в классе следующие методы: 
1) конструктор без параметров (открывает файл с именем test_text.txt и считывает из него данные по отдельным словам в массив строк) 
2) конструктор с параметрами 
(принимает количество слов, которое должно быть считано из файла test_text.txt, и считывает это количество слов в массив строк) 
3) конструктор с параметрами (принимает имя текстового файла и считывает из него данные по отдельным словам в массив строк) 
4) конструктор с параметрами (принимает имя текстового файла и количество слов, которое должно быть считано из этого файла в массив строк) 
5) конструктор копирования 
6) конструктор перемещения 
7) деструктор 
8) get_max_len (возвращает максимальную длину слова, содержащегося в тексте) 
9) get_max_word (возвращает первое слово с максимальной длиной, содержащееся в тексте)  
Протестируйте работу класса.
