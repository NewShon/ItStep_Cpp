Задание  
А. Создайте иерархию геометрических фигур, в которой должны быть следующие классы:  
родительский класс Shape c одним чисто виртуальным методом getArea.  
класс Triangle, который будет унаследован от класса Shape. 
В этом классе должны быть поля для хранения трех сторон треугольника и трех углов. 
Предусмотрите разные конструкторы для класса: 
а) по трем сторонам; 
б) по двум сторонам и углу между ними; 
в) по стороне и двум углам, прилежащим к ней. 
Реализуйте аксессоры и метод getArea для вычисления площади треугольника.  
класс Parallelogram, который будет унаследован от класса Shape. 
В этом классе должны быть поля для хранения двух сторон параллелограмма и одного угла. 
Реализуйте конструктор с параметрами, аксессоры и метод getArea.  
класс Rectangle, который будет унаследован от класса Parallelogram. 
Реализуйте конструктор с параметрами, аксессоры и метод getArea.  
класс Square, который будет унаследован от класса Rectangle. 
Реализуйте конструктор с параметрами, аксессоры и метод getArea.  
класс Circle, который будет унаследован от класса Shape. 
Предусмотрите поля для хранения радиуса и центра круга. 
Реализуйте конструктор с параметрами, аксессоры и метод getArea. !!! 
При создании заголовочных файлов используйте формат Фамилия_Фигура. 
Протестируйте написанные классы.
