 Задание №2  
В контейнере std::vector хранится набор целых чисел. 
Решите следующие задачи с помощью лямбда-функций и алгоритмов STL:  
заменить в векторе все числа, кратные 10, на 11;  
отсортировать данные по убыванию;  
сохранить в новый вектор только те значения, которые больше переданного значения v в n раз;  
увеличить каждое значение в контейнере на заданную константу n, которая передается извне, и сохранить новые значения в другом векторе;  
найти в векторе самое большое число, кратное n; 
найти в векторе самое маленькое число из тех, которые больше n;  
отсортировать данные так, чтобы первыми пятью элементами стали пять наибольших чисел, 
содержащихся в векторе, и чтобы они были расположены по убыванию (остальные элементы не нужно сортировать); 
заменить в векторе все числа, которые начинаются с двойки, на 0.
После каждой позиции выведите в консоль данные из вектора.
Протестируйте работу функций на разных числовых данных.
