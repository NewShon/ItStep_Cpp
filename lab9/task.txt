ЛАБОРАТОРНАЯ РАБОТА №9 
 
Задание 
 Создайте клас Printer «Принтер».
 Класс должен содержать поля с информацией о печатающем устройстве 
 (модель, год выпуска, производитель, текущий документ для распечатки и другие, которые покажутся вам необходимыми).
 В класс Printer добавьте конструктор с параметрами.
 Обеспечьте возможность вести подсчет того,
 сколько всего документов было напечатано на каждом отдельном принтере и на всех принтерах вместе. 
 Реализуйте следующие методы: 
 
 printCopy (переписывает содержимое из одного файла в другой); 
 printCharsWithStep (печатает через пробел каждый n-ый символ из входного файла, где n передается как параметр метода);  
 printWordsWithStep (печатает через пробел каждое n-ое слово из входного файла, где n передается как параметр метода);  
 printReverse (выводит содержимое файла в обратном порядке посимвольно);  
 printСoncordance (принимает слово для поиска, и если оно есть в файле, 
 то печатает контекст с этим словом – по три слова справа и слева от заданного).
 Протестируйте работу приложения.
