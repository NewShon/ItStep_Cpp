Создайте класс Area для подсчета площади геометрических фигур. Класс должен предоставлять функциональность для подсчета площади следующих фигур:
1) прямоугольник
a) вычисление площади по двум сторонам
b) вычисление площади по одной стороне, если прямоугольник является квадратом
2) параллелограмм
a) вычисление площади по одной стороне и высоте
b) вычисление площади по двум сторонам и углу между ними
3) ромб
a) вычисление площади по стороне и углу
b) вычисление площади по двум диагоналям
Функции-члены для подсчета площади должны быть реализованы как статические. Также класс должен считать, сколько раз уже вычислялась площадь с его помощью, и возвращать это значение через статическую функцию-член.
Дополнительные указания:
1) объявление класса и реализацию методов храните в разных файлах;
2) все имена функций образуйте по модели get‹НазваниеФигуры›Area;
3) следите за использованием модификатора const при аргументах методов;
4) не забудьте, что все стороны и углы должны быть неотрицательными;
5) протестируйте написанный класс на корректных и некорректных входных данных.
