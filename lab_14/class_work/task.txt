Работа с контейнерами   
Выполните следующие задания. 
1. Считайте пословно текст из файла turgenev.txt. 
2. Запишите в файл turgenev_output.txt количество слов, содержащихся во входном файле. 
3. Извлеките из текста все уникальные слова, длина которых равна 7 символов. 
Получившееся количество слов запишите в файл turgenev_output.txt. 
4. Из полученного списка выберите все слова, заканчивающиеся на -ти, -ть, -чь, и сохраните в новом контейнере Inf. 
Запишите получившееся количество слов и сами слова в файл (разделитель между словами – пробел). 
5. Удалите из контейнера Inf два первых слова. 
6. Удалите из контейнера Inf слова под номерами 9, 17, 35, 40, 41, 42, 46, 51 (идексация начинается с 1). 
7. Удалите из контейнера Inf слова полночь, прихоть, страсть. 
8. Удалите из контейнера Inf слова, начинающиеся на рад-, роб-, сов-, тяг-, тяж-. 
9. Добавьте в начало контейнера Inf слово акать, а в конец – слово якать. 
10. Вставьте перед словом обедать слово начать. Запишите получившийся список слов в файл.
