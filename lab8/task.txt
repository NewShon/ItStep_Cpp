Лабораторная работа №8
Задание
Создайте класс Complex для работы с комплексными числами. Комплексное число – это
число вида x + yi, где x и y – это вещественные числа, а i – мнимая единица (i2 = -1).
Число x – это действительная часть комплексного числа, а y – мнимая часть.
А. Реализуйте следующие методы:
- конструктор по умолчанию;
- конструктор с двумя параметрами (принимает действительную и мнимую части);
- конструктор копирования;
- оператор копирующего присваивания;
- деструктор.
Б. Перегрузите следующие операторы:
- оператор + (складывает два числа типа Complex);
- оператор - (вычитает два числа типа Complex);
- оператор * (умножает два числа типа Complex);
- оператор / (делит два числа типа Complex);
- составные операторы +=, -=, *=, /=;
- операторы ++ и -- (увеличивают или уменьшают );
- операторы сравнения ==, != , > , < , >= , <= (сравнивают два числа типа Complex);
- оператор ( ) (изменяеют действительную и мнимую часть комплексного числа);
- оператор << (выводит комплексное число в консоль);
- оператор >> (вводит комплексное число из консоли);
В. Добавьте операторы +, –, * и /, которые будут принимать одно комплексное число, а
другое – вещественное, причем в любом порядке.
